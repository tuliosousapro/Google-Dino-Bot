## Contributing Guidelines

### 🦖 Purpose of This Project  
Google‑Dino‑Bot is a lightweight, experimental automation script built for educational use. Contributions should keep the project simple, fast, and easy to understand.

---

### 📌 What We Accept  
- Improvements to code clarity, structure, or performance  
- Bug fixes related to obstacle detection or key events  
- Enhancements that keep the bot lightweight and dependency‑free  
- Documentation improvements (README, comments, examples)  
- Security or compatibility fixes for chromedino.com  

---

### 🚫 What We Do NOT Accept  
- Features that add heavy dependencies  
- Code that interacts with or automates YouTube, Google, or other platforms  
- Attempts to bypass platform protections or violate Terms of Service  
- Obfuscated, minified, or unclear code  
- AI‑generated PRs with no human review  

---

### 🧪 How to Contribute  
1. **Fork** the repository  
2. **Create a new branch** for your change  
3. Make your edits with clear, readable code  
4. Ensure the bot still runs correctly on [chromedino.com](https://chromedino.com)  
5. Submit a **Pull Request** with:  
   - A clear description of the change  
   - Before/after behavior if applicable  
   - Screenshots or GIFs for visual changes  

---

### 📝 Code Style  
- Keep everything minimal and readable  
- Use consistent indentation  
- Add comments only where needed  
- Avoid unnecessary abstractions  

---

### 🔍 Testing  
Before submitting a PR, verify:  
- The bot jumps correctly for both high and low obstacles  
- No console errors appear  
- The script works in a clean Chrome session  

---

### 🤝 Community Standards  
Be respectful, constructive, and focused on improving the project. Low‑effort or spammy PRs will be closed.
